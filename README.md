# English

A modern implementation of the classic Snake game with mobile swipe controls and desktop keyboard support.

## 🎮 Features

- 📱 Mobile-friendly with swipe controls
- 💻 Desktop support with keyboard arrows
- 🎨 Responsive design
- 🌍 Multi-language support (EN/TR)
- 🎵 Sound effects
- 🎯 Progressive difficulty
- 💨 Smooth animations
- 🌓 Frosted glass effects

## 🛠️ Technologies Used

- React
- i18next (Multi-language support)
- Web Audio API (Sound effects)
- CSS3 (Modern styling and animations)

## ⚙️ Installation

1. Clone the repository:git clone https://github.com/emreicoz07/snake-game.git

2. Navigate to project directory:cd snake-game

3. Install dependencies:npm install

4. Run the project:npm start

5. Run the project with mobile:npm run start-mobile

## 🎯 How to Play

### 💻 On Desktop:
- Use arrow keys to control the snake
- Press Enter to restart the game

### 📱 On Mobile:
- Swipe in any direction to control the snake
- Tap screen to restart when game is over

### 🎮 General Rules:
- Collect food to grow and score points
- Avoid hitting walls and snake's tail
- Snake speeds up with each food collected
